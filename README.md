# Cathedrale_control

Contrôleur de séquence LED pour ESP32-H2 via Bluetooth Low Energy (BLE).
Le controlleur est ICI a Utiliser avec Google Chrome : https://julienrat.github.io/Cathedrale_control/
## Fonctionnalités
- Interface web interactive pour créer des séquences lumineuses.
- Gestion intelligente des 6 canaux PWM de l'ESP32-H2 (permet de contrôler 8 LEDs).
- Déclenchement par bouton physique sur GPIO 22.
- Sauvegarde de la séquence en mémoire Flash.
